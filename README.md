# Financial-Performance-Dashboard-Power-BI-Project

## Overview
This Power BI dashboard compares financial performance between 2024 and 2025, highlighting revenue growth, expense trends, and profitability changes.

## Key Insights
- Revenue increased by 9% YoY.
- Expenses rose due to HR, IT, Rent, and Marketing costs.
- Net Profit improved despite higher expenses.
- Business growth remained stable across both years.

## Tools Used
- Power BI
- DAX
- Excel
